Системные требования.

Для работы модуля, Ваш хостинг должен поддерживать:

  • Ioncube Loader версии 5.0 и выше;
  • PHP версии 5.4 и выше;

Для интеграции функционала модуля и движка магазина версий 2.x, 3.x используется ocmod. Для 1.5.x используется vqmod.

Совместимость.

Работает на OpenCart: 1.5.x (до версии 3.1.0), 2.0.x, 2.1.x, 2.2.x, 2.3.x, 3.0.x

Работает на ocStore: 1.5.x (до версии 3.1.0), 2.1.x, 2.3.x, 3.0.x

Работает на OpenCart.Pro, ocShop и других сборках на базе OpenCart соответствующих версий.

  • русский
